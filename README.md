# 🤖 Automação de Cadastro de Produtos com Python

Este projeto foi desenvolvido durante o **Intensivão de Python da Hashtag Programação**.  
O objetivo é **automatizar o processo de login e cadastro de produtos em um sistema web** utilizando **RPA (Robotic Process Automation)**.

---

## 📝 Descrição

O script utiliza a biblioteca **PyAutoGUI** para controlar **mouse e teclado**, realizando o **preenchimento automático de um formulário web** a partir de uma base de dados em **.csv**.

---

## ⚙️ Funcionalidades

- 🌐 **Abertura automática do navegador** (configurado para Opera)
- 🔐 **Login automatizado** no sistema
- 📊 **Leitura de base de dados** utilizando a biblioteca **Pandas**
- 🔁 **Loop de cadastro automático**
  - Preenche todos os itens da tabela
  - Lida com **campos opcionais**

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **PyAutoGUI** → Automação de interface (mouse e teclado)
- **Pandas** → Manipulação e leitura de dados
- **Time** → Controle de intervalos para carregamento do sistema

---

## 💡 Conceitos aplicados

- Automação de tarefas repetitivas
- Manipulação de dados com **DataFrame**
- Integração entre **RPA e análise de dados**
- Uso de **loops** para processamento em massa
